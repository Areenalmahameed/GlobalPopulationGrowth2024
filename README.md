# GlobalPopulationGrowth2024
Global Population Growth 2024 - Clustering Analysis
By: Areen Al-Mahameed
Tools & Libraries
`Python`, `Pandas`, `NumPy`, `Scikit-learn`, `Optuna`, `Matplotlib`, `Seaborn`, `Plotly`
Steps
1. Preprocessing: StandardScaler + OneHotEncoder  
2. Optimization: Optuna to tune `n_clusters` and `init` method  
3. Clustering: Final KMeans model trained with best parameters  
4. Visualization: Plotly (interactive) + Matplotlib (static)
5. Result
The model groups cities by **population growth patterns** between 2023–2024  
and visualizes clusters clearly using color-coded charts
